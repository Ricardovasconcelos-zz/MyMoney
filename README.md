### Backend

module.exports = {

  // JWT KEY
  
  authSecret: 'yourkeyhere'
  
}

Run ( npm run dev ) to init backend

### Frontend
Run ( npm run dev ) to init frontend

Open http://localhost:8080 to view it in the browser.
